# Production Art Guard

Use this skill when:
- working on ManorWorldStage
- replacing environment art
- replacing character art
- changing camera direction or pacing
- changing inspection/surveillance presentation
- doing showcase-polish passes

Rules:
1. Think like a game art director, not a dashboard engineer.
2. Prioritize readability, composition, and hierarchy.
3. Preserve gameplay semantics.
4. Preserve benchmark safety.
5. Preserve legal cleanliness of assets.
6. Favor one strong visual improvement over many weak ones.

Before finishing, verify:
- whole-manor readability improved
- room differentiation improved
- character readability improved if in scope
- camera and pacing still support spectator understanding
- no new benchmark leakage was introduced

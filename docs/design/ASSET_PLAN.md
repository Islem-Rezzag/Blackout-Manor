# Asset Plan

This plan separates what is safe to import now, what is reference-only, and what should be replaced by bespoke Blackout Manor assets later.

## Importable Now

### Room And Environment
- OGA `Modern Houses Tileset TopDown` (CC0)
  - use only as a safe blocker baseline for floorplan readability, room dressing placeholders, and temporary cutaway support

### UI
- Kenney UI Pack (CC0)
  - menu buttons
  - shell prompts
  - generic secondary panels outside the live runtime

### Audio
- Kenney UI Audio (CC0)
  - launcher clicks
  - confirm and back sounds
  - basic shell prompts

## Reference-Only For Mood

### Environment Mood
- OGA `mansion interior top down tileset` (CC-BY-SA 4.0)
- LPC or LPC-derived interior references

### Genre Mood
- OpenSuspect
- film and illustration references for storm-noir, masquerade, and gothic domestic tension

### Motion
- `kumarankm/Amongus` for locomotion cadence study only

## License Notes
- Prefer MIT and CC0 imports for anything that actually lands in the repo
- Track every imported asset source in a future credits ledger before public release
- Treat CC-BY-SA and mixed-license packs as reference-only unless later cleared by a dedicated license review
- Code remains MIT; imported art/audio must be tracked separately from code licensing

## Attribution Expectations
- CC0 assets do not require attribution, but voluntary credits are still recommended
- MIT code donors should be acknowledged in docs
- Reference-only sources should be listed as inspiration, not as imported assets

# Art Replacement Priority

## Goal
This ranking orders replacement work by maximum visual impact on the current spectator runtime. The ordering is not about implementation convenience. It is about what most quickly changes the impression from "strong alpha" to "real game."

## Priority Ranking

### 1. Room shells
Replace first.

Why:
- the manor shell language dominates the whole-house read
- every room currently inherits too much of the same procedural framing vocabulary
- if shells stay placeholder-grade, the house will keep reading like a system demo no matter how much other polish lands

Impact:
- immediately changes the poster-quality of `/game`
- makes the estate feel authored instead of generated

## 2. Floors / walls / thresholds
Replace second.

Why:
- material repetition is one of the fastest tells that the environment is still baseline-driven
- thresholds are essential to travel readability and house identity
- walls and floors carry the mood split between warm manor interior and cold storm edge

Impact:
- upgrades room identity even before secondary dressing is complete
- improves realism, material richness, and connected-house credibility

## 3. Hero props
Replace third.

Why:
- current rooms rely too heavily on one sparse task prop plus abstract decor
- hero props are the fastest way to make rooms feel inhabited and distinct
- task readability will improve if the props look like real room anchors instead of placeholder silhouettes

Impact:
- sharpens room fiction
- improves screenshot value and spectator memory of each space

## 4. Table / meeting set
Replace fourth.

Why:
- meetings are among the most dramatic spectator moments
- the current blocking logic is strong, but the set itself is not yet iconic
- a premium long table, chair set, lighting package, and room backdrop will raise the perceived quality of the whole game

Impact:
- large dramatic upgrade for reports, meetings, votes, and reveals
- high-value visual payoff in trailers, captures, and showcase footage

## 5. Portraits
Replace fifth.

Why:
- portraits are close-up identity moments
- current portrait cards are orderly and readable, but not yet premium character art
- better portraits will increase cast attachment and help the game feel authored

Impact:
- strong payoff in meetings and marketing captures
- improves cast identity without yet solving in-world character quality by itself

## 6. Character sprites / rigs
Replace sixth.

Why:
- the current procedural rig has done its job, but it is still visibly an alpha solution
- final character quality depends on authored silhouettes, garments, and motion
- this is high impact, but it comes after environment replacement because the environment currently sets the dominant first impression

Impact:
- transforms cast presence and movement credibility
- critical for final-art confidence

## 7. Subtitle / event-board typography
Replace seventh.

Why:
- current UI hierarchy is solid, but the typography system is not yet a distinctive final broadcast package
- UI refinement will matter more after the world and cast art are stronger

Impact:
- sharpens brand identity
- helps the runtime look premium instead of merely well organized

## 8. Ambient VFX / audio
Replace eighth.

Why:
- current ambience and cue logic are useful but still placeholder-safe
- audio and VFX should be upgraded after the world and cast art establish the final visual identity

Impact:
- deepens atmosphere and polish
- raises production feel, but should not be mistaken for a substitute for better environment and character art

## Replacement Order Summary
1. room shells
2. floors / walls / thresholds
3. hero props
4. table / meeting set
5. portraits
6. character sprites / rigs
7. subtitle / event-board typography
8. ambient VFX / audio

## Rule Of Thumb
If only one replacement wave can happen before a bigger showcase, do environment first. The environment currently carries the largest share of the "still alpha" read.

# Blackout Manor Roadmap

This roadmap is the top-level implementation sequence for the project.

## Build Philosophy

Preserve the strongest system spine:
- deterministic engine
- replay
- fairness and EQ analysis
- benchmark safety
- world-first runtime

But do not let strong systems excuse weak presentation.
The goal is a game-first spectator experience and a trustworthy EQ benchmark at the same time.

## Milestones

### M1 - Route and product-surface inversion
Make the live route game-first and push replay and fairness out of the default live path.

### M2 - World-first runtime
Replace page-first simulation with a real runtime-owned spectator surface.

### M3 - Surveillance and observation
